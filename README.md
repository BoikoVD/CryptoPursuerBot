# My CryptoPursuer telegram bot.

## Version 1.

#### Bot send every 60 sec messages with coins name which grew up on 5%.
#### You can choose coins which want to pursuering, adding their symbol to array "pursueringCoins". Example:

let pursueringCoins = [
	"BTC",
	"ETH",
	"BNB",
	"ICP",
	"SOL",
];
